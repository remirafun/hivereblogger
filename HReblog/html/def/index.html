Hive Reblogger

Hive Reblogger is a tool to automatically reblog new posts which fit your predefined criteria. For example, you might'd like to reblog new posts voted for by a given author. With this tool you can do just that.

Features

  • reblog a post voted by selected author/s
  • reblog a new post filtered by authors, tags, keywords, etc...
  • is there a feature you'd like missing? Let us know about it!

Open Source

Hive Reblogger is open source. You can access its repository here.
